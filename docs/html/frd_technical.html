Architecture Overview

ChessIQ uses a layered intelligence architecture that separates engine analysis (Stockfish) from pattern recognition and AI coaching. This enables reusability of Stockfish analysis for pattern detection, scalable pattern computation across large game histories, and context-aware AI coaching that synthesizes multiple data sources.

1.1 Stack

LayerTechnology
FrontendNext.js (React, TypeScript), React Query, Axios, TailwindCSS — AI Coach chat, pattern visualization, interactive training board
API GatewayFastAPI, SQLAlchemy, Pydantic v2

Background WorkersCelery (distributed analysis and pattern detection)

PGN + Stockfish Analysis Pipeline

4.1 Pipeline Overview

  1. Load games.pgn
  2. Parse PGN into move list using python-chess
  3. For each position before a move: evaluate with Stockfish, determine best move, compute eval delta
  4. Classify move quality (good/inaccuracy/mistake/blunder)
  5. Aggregate stats per game and by phase
  6. Store results in game_analyses; mark game is_analyzed = true

4.2.3 Move Quality Classification

ClassificationEval Delta (centipawns)
Blunder≥ 200 cp or losing forced mate
Mistake≥ 100 cp
Inaccuracy≥ 50 cp
GoodOtherwise

Where eval_delta = eval_after_best - eval_after_user (engine POV).

4.2.4 ACPL & Accuracy

  • ACPL = average of abs(eval_after_user - eval_before) across all user moves
  • Accuracy mapping: ACPL <20 → 95–100 · 20–50 → 80–95 · 50–100 → 60–80 · >100 → <60

4.2.5 Phase-Based Metrics

  • Opening: moves 1–12
  • Middlegame: moves 13–30
  • Endgame: moves >30 or when material is sufficiently reduced

Game Fetching & Caching

5.1 Chess.com Fetch Strategy

  • Endpoint: https://api.chess.com/pub/player/{username}/games/{YYYY}/{MM}
  • Fetch in reverse chronological order until N games collected or date range exhausted

5.2 Redis Caching Layer

  • chesscom:archives:{username}:{year}:{month} → archived JSON/PGN (TTL 1h)
  • chesscom:rate:{username} → request counter per minute
  • On fetch: check rate key → check archive key → call Chess.com → cache result

5.3 Replace-All Semantics

  1. Start DB transaction
  2. Delete game_analyses for user_id
  3. Delete games for user_id
  4. Insert fetched games
  5. Update users.total_games and users.last_fetch_at
  6. Commit

class TacticalPatternDetector:
    def detect(self, user_analyses: List[GameAnalysis]) -> List[Pattern]:
        patterns = []
        theme_counts = defaultdict(list)

        for analysis in user_analyses:
            for moment in analysis.critical_moments:
                if moment.error_type in ['blunder', 'mistake']:
                    theme = self.classify_tactical_theme(moment)
                    if theme:
                        theme_counts[theme].append(moment)

        for theme, occurrences in theme_counts.items():
            if len(occurrences) >= MIN_PATTERN_THRESHOLD:
                confidence = self.calculate_confidence(occurrences, len(user_analyses))
                patterns.append(Pattern(
                    type='tactical', subtype=theme,
                    occurrences=len(occurrences),
                    confidence=confidence, examples=occurrences[:5]
                ))
        return patterns

Opening Pattern Detector

python
class OpeningPatternDetector:
    def detect(self, user_analyses: List[GameAnalysis]) -> List[Pattern]:
        opening_performance = defaultdict(lambda: {'games': [], 'results': []})

        for analysis in user_analyses:
            eco = analysis.opening_eco
            opening_performance[eco]['games'].append(analysis)
            opening_performance[eco]['results'].append(analysis.game.result)

        for eco, data in opening_performance.items():
            games, results = data['games'], data['results']
            if len(games) >= MIN_OPENING_GAMES:
                win_rate = results.count('win') / len(results)
                if win_rate < 0.35:
                    yield Pattern(type='opening', subtype=eco,
                                  win_rate=win_rate, is_strength=False)
                elif win_rate > 0.7:
                    yield Pattern(type='opening', subtype=eco,
                                  win_rate=win_rate, is_strength=True)

7.3 Pattern Confidence Scoring

python
def calculate_pattern_confidence(
    occurrences: int, total_games: int, last_seen_days: int = None
) -> float:
    sample_confidence = min(occurrences / 10, 1.0)
    prevalence_score  = min((occurrences / total_games) * 3, 1.0)

    recency_multiplier = 1.0
    if last_seen_days:
        if last_seen_days > 90: recency_multiplier = 0.5
        elif last_seen_days > 30: recency_multiplier = 0.8

    return round(
        (sample_confidence * 0.4 + prevalence_score * 0.6) * recency_multiplier, 2
    )

Move Timing & Behavioral Analysis NEW

8.1 Time Data Extraction

python
class MoveTimingExtractor:
    def extract_from_pgn(self, pgn: str, user_color: str) -> List[MoveTiming]:
        game = chess.pgn.read_game(io.StringIO(pgn))
        timings = []
        prev_clock = None

        for move_number, node in enumerate(game.mainline(), start=1):
            clock_str = node.clock()
            if clock_str is not None:
                clock_remaining = self.parse_clock(clock_str)
                think_time = (prev_clock - clock_remaining) if prev_clock else None
                phase = self.determine_phase(move_number, node.board())
                time_pressure = self.is_time_pressure(clock_remaining, game.time_control)

                timings.append(MoveTiming(
                    move_number=move_number,
                    think_time_seconds=think_time,
                    clock_remaining_seconds=clock_remaining,
                    time_pressure=time_pressure,
                    phase=phase
                ))
                prev_clock = clock_remaining
        return timings

8.2 Behavioral Pattern Detection

python
class TimingPatternDetector:
    def detect(self, user_timings: List[MoveTiming]) -> TimingPatterns:
        patterns = TimingPatterns()

        long_thinks   = [t for t in user_timings if t.think_time_seconds > 120]
        quick_moves   = [t for t in user_timings
                         if t.think_time_seconds < 5 and not t.is_book_move]
        pressure_moves = [t for t in user_timings if t.time_pressure]

        patterns.overthink_tendency   = len(long_thinks)    / len(user_timings)
        patterns.impulsive_tendency   = len(quick_moves)    / len(user_timings)
        patterns.time_pressure_ratio  = len(pressure_moves) / len(user_timings)

        phase_times = defaultdict(list)
        for t in user_timings:
            phase_times[t.phase].append(t.think_time_seconds)

        patterns.phase_time_distribution = {
            phase: mean(times) for phase, times in phase_times.items()
        }
        return patterns

class CoachContextAssembler:
    def assemble_context(self, user_id, user_message, conversation_history):
        context = CoachContext()
        context.player_profile = self.db.get_player_profile(user_id)
        all_patterns = self.db.get_player_patterns(user_id)
        context.relevant_patterns = self.filter_patterns_by_query(all_patterns, user_message)

        if self.is_game_specific_query(user_message):

        context.query_type = self.classify_query(user_message)
        return context

Updated Recommendation Logic

10.2 Pattern-Based Recommendations

Priority order: Critical patterns → Significant patterns → Strength building

python
def build_recommendations_from_patterns(patterns, profile):
    sorted_patterns = sorted(
        patterns, key=lambda p: (p.severity != 'critical', -p.confidence_score)
    )
    recs = []
    for pattern in sorted_patterns[:7]:
        if pattern.is_strength:
            rec = Recommendation(
                category=f"Leverage Your {pattern.pattern_type.title()} Strength",
                title=f"You're strong at {pattern.pattern_subtype}",
                severity="positive", recommendation_type="strength_building",
                confidence=pattern.confidence_score
            )
        else:
            rec = Recommendation(
                category=pattern.pattern_type.title(),
                title=f"Work on {pattern.pattern_subtype}",
                severity=pattern.severity, recommendation_type="weakness_correction",
                drill_type=pattern.recommended_drill_type,
                confidence=pattern.confidence_score
            )
        recs.append(rec)
    return recs

Malformed PGNsParse exceptionLog error with game id, skip game, mark invalid_pgn, continue batch
Chess.com Rate LimitHTTP 429Set Redis cooldown for username, return 503 to frontend with friendly message
Supabase / DB IssuesConnection errorExponential backoff, log, return 503: "Service temporarily unavailable"
LLM Service Failure NEWHTTP 429/500 from providerReturn cached fallback; show "AI Coach temporarily unavailable"; queue for retry
Pattern Detection Failure NEWCelery task failure after max retriesLog failure; still show Stockfish results; schedule retry in 1 hour; alert engineering
