Architecture Overview

ChessIQ implements a three-layer intelligence architecture: Stockfish for deterministic chess analysis (position evaluation, move quality, tactical detection), a Pattern Recognition Engine for longitudinal behavioral analysis, and an AI Coach Service for personalized conversational coaching. The LLM's role is strictly translation and explanation — never raw chess reasoning — enabling a local-first inference model that achieves 90–98% cost reduction versus hosted-only architectures.
